# 材料管理系统 - 安装指南（小白专用）

## 📦 软件文件说明

打包完成后，你会得到以下文件：

```
材料管理系统/
├── dist/
│   └── 材料管理系统.exe    ← 这就是主程序！
├── README.txt              ← 使用说明
└── icon.ico                ← 程序图标
```

**你只需要把 `dist` 文件夹里的 `材料管理系统.exe` 复制到目标电脑即可！**

---

## 🖥️ Windows系统安装方法

### 方法一：直接运行（最简单）

1. **找到程序文件**
   - 在文件夹中找到 `材料管理系统.exe`

2. **双击运行**
   - 像打开其他软件一样，双击这个文件
   - 第一次运行可能需要几秒钟启动

3. **完成！**
   - 程序会自动创建数据文件夹
   - 数据保存在：`C:\Users\你的用户名\.document_manager\`

### 方法二：创建桌面快捷方式（推荐）

1. **右键点击** `材料管理系统.exe`
2. **选择** "发送到" → "桌面快捷方式"
3. **以后直接在桌面双击图标即可**

### ⚠️ 可能遇到的问题

**问题1：提示"无法运行"或"被阻止"**
- 原因：Windows安全设置
- 解决方法：
  1. 右键点击程序 → 属性
  2. 勾选"解除锁定"（如果有）
  3. 点击确定，再试一次

**问题2：杀毒软件报警**
- 原因：杀毒软件不认识这个新程序
- 解决方法：
  1. 选择"允许"或"添加信任"
  2. 或暂时关闭杀毒软件再运行

**问题3：双击没反应**
- 解决方法：
  1. 右键点击 → 以管理员身份运行
  2. 或检查是否有其他程序冲突

## 🐧 国产操作系统安装方法（统信UOS/麒麟OS）

### 准备工作

首先，你需要确认你的系统类型：
- **统信UOS**：桌面底部有"UOS"标志
- **麒麟OS**：桌面底部有"麒麟"标志
- **深度Deepin**：桌面底部有"Deepin"标志

### 方法一：使用PyInstaller打包的文件

#### 第一步：复制文件到目标电脑

1. 准备一个U盘
2. 把 `材料管理系统`（Linux版本）复制到U盘
3. 把U盘插到国产电脑上
4. 把文件复制到电脑的"文档"或"桌面"文件夹

#### 第二步：设置文件权限

**方法A：使用鼠标（图形界面）**
1. 右键点击 `材料管理系统` 文件
2. 选择"属性"
3. 点击"权限"标签
4. 勾选"允许作为程序执行"
5. 点击确定

**方法B：使用终端（命令行）**
1. 打开终端（按 Ctrl+Alt+T，或在应用菜单找"终端"）
2. 输入以下命令（把"用户名"换成你的）：
   ```bash
   cd /home/用户名/文档
   chmod +x 材料管理系统
   ```
3. 按回车执行

#### 第三步：运行程序

**方法A：双击运行**
- 直接双击 `材料管理系统` 文件
- 选择"运行"或"执行"

**方法B：终端运行**
```bash
./材料管理系统
```

### 方法二：使用AppImage文件（推荐）

AppImage是一种特殊的文件格式，双击就能运行，不需要安装。

#### 第一步：复制文件

1. 把 `材料管理系统-x86_64.AppImage` 复制到目标电脑

#### 第二步：设置权限

```bash
chmod +x 材料管理系统-x86_64.AppImage
```

#### 第三步：运行

双击文件，或输入：
```bash
./材料管理系统-x86_64.AppImage
```

## 🔧 常见问题解答（FAQ）

### Q1：程序打不开怎么办？

**Windows：**
1. 右键 → 以管理员身份运行
2. 检查杀毒软件是否拦截
3. 确认是64位系统（右键"此电脑"→属性查看）

**国产系统：**
1. 确认已添加执行权限（chmod +x）
2. 检查是否安装了必要的系统库
3. 尝试在终端运行查看错误信息

### Q2：提示缺少字体或显示乱码？

**国产系统解决方法：**
```bash
# 安装中文字体
sudo apt install fonts-wqy-zenhei fonts-wqy-microhei

# 刷新字体缓存
fc-cache -fv
```

### Q3：数据存在哪里？

**重要说明：** 程序的数据文件会自动保存在用户目录下，与程序文件分开存储，这样：
- 程序更新不会丢失数据
- 可以把程序放在任意位置
- 数据安全有保障

**数据存储位置：**
- **Windows：** `C:\Users\你的用户名\AppData\Roaming\document_manager\`
  - 或直接按 Win+R，输入 `%APPDATA%\document_manager` 回车
- **国产系统：** `/home/你的用户名/.document_manager/`

**数据目录结构：**
```
document_manager/
├── database.db        ← 数据库文件（文档、金句等）
├── config/            ← 配置文件
│   ├── settings.json
│   └── rules.json
├── documents/         ← 导入的文档
├── backups/           ← 备份文件
└── logs/              ← 日志文件
```

### Q4：如何备份数据？

1. 打开程序 → 系统设置 → 备份管理
2. 点击"立即备份"
3. 备份文件会保存在你指定的位置

### Q5：如何迁移到新电脑？

1. 在旧电脑上备份数据
2. 复制程序文件和备份文件到新电脑
3. 在新电脑上运行程序
4. 使用"恢复备份"功能

## 📞 需要帮助？

如果遇到问题：

1. **查看日志文件**
   - Windows: `C:\Users\用户名\.document_manager\logs\`
   - 国产系统: `~/.document_manager/logs/`

2. **检查系统要求**
   - 内存：4GB以上
   - 硬盘：500MB以上
   - 操作系统：Windows 7+ 或 国产Linux

3. **重新下载**
   - 如果文件损坏，重新获取程序文件
