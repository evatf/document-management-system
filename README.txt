                          材料管理系统 使用说明

================================================================================

一、软件简介
--------------------------------------------------------------------------------
材料管理系统是一款专为党政机关设计的文档管理软件，主要功能包括：

  1. 材料库：文档导入、分类管理、标签管理、全文检索
  2. 文档检索：多关键词搜索、标签筛选、高亮显示
  3. 智能分析：词频分析、文稿比对、相似度检测、句式提取
  4. 星标收藏：重要文档收藏、分类管理
  5. 金句库：金句收藏、分类管理、快速检索
  6. 系统设置：数据备份、数据清理、系统配置


二、系统要求
--------------------------------------------------------------------------------
【Windows系统】
  - 操作系统：Windows 7/8/10/11
  - 内存：4GB以上（推荐8GB）
  - 硬盘：500MB以上可用空间

【国产操作系统】
  - 统信UOS 20/21
  - 麒麟OS V10
  - 深度Deepin 20/23
  - 内存：4GB以上（推荐8GB）
  - 硬盘：500MB以上可用空间


三、安装说明
--------------------------------------------------------------------------------
【Windows系统】
  1. 双击"材料管理系统.exe"即可运行
  2. 首次运行会在用户目录创建数据文件夹

【国产操作系统】
  1. 打开终端，进入程序所在目录
  2. 添加执行权限：chmod +x 材料管理系统
  3. 运行程序：./材料管理系统
  4. 或双击"材料管理系统"文件运行

【AppImage版本】
  1. 添加执行权限：chmod +x 材料管理系统-x86_64.AppImage
  2. 双击运行或：./材料管理系统-x86_64.AppImage


四、功能说明
--------------------------------------------------------------------------------
【材料库】
  - 支持导入Word(.docx)、Excel(.xlsx)、PDF(.pdf)、TXT格式文档
  - 支持文件夹批量导入
  - 支持文档分类、标签管理
  - 支持文档预览和编辑

【文档检索】
  - 支持多关键词搜索（空格分隔）
  - 支持按分类、年份、标签筛选
  - 搜索结果高亮显示关键词

【智能分析】
  - 词频分析：统计高频词汇，生成词云
  - 文稿比对：对比两篇文档差异，同步滚动显示
  - 相似度检测：检测文档相似程度
  - 句式提取：提取常用句式和排比句

【星标收藏】
  - 收藏重要文档
  - 支持分类管理

【金句库】
  - 收藏常用金句
  - 支持分类管理
  - 支持从文档预览中直接收录

【系统设置】
  - 数据备份：手动/自动备份
  - 数据清理：清除文档、金句等数据
  - 系统配置：存储路径、备份设置


五、数据存储
--------------------------------------------------------------------------------
数据默认存储位置：
  - Windows: C:\Users\[用户名]\.document_manager\
  - Linux: ~/.document_manager/

数据文件：
  - database.db：SQLite数据库文件
  - backups/：备份文件夹


六、常见问题
--------------------------------------------------------------------------------
【问题1】程序无法启动
  - 检查是否有足够的系统权限
  - 检查是否有杀毒软件拦截

【问题2】界面显示乱码
  - Linux系统请安装中文字体：
    sudo apt install fonts-wqy-zenhei fonts-wqy-microhei

【问题3】无法导入文档
  - 检查文档格式是否支持
  - 检查文档是否损坏

【问题4】数据丢失
  - 检查备份文件夹是否有备份
  - 使用系统设置的恢复功能


七、技术支持
--------------------------------------------------------------------------------
如遇问题，请检查日志文件：
  - Windows: C:\Users\[用户名]\.document_manager\logs\
  - Linux: ~/.document_manager/logs/
